This is a scrollable text container with IntersectionObserver-based fade effects.

As you scroll, each paragraph fades in and slides up as it enters the visible area. This creates a dynamic, engaging reading experience.

The fade effect uses the IntersectionObserver API to detect when elements enter the viewport, then triggers CSS transitions.

Unlike CSS masking, which creates a static window effect, this approach animates each element individually as it becomes visible.

The implementation is pure CSS transitions triggered by JavaScript - no complex animation libraries needed.

You can easily adjust three key parameters in the CSS:

1. Slide Distance: How far elements start below their final position (currently 20px).

2. Fade Duration: How long the opacity transition takes (currently 800ms).

3. Slide Duration: How long the slide-up animation takes (currently 600ms).

The scrollbar is styled to be minimal and unobtrusive, matching modern design patterns.

How to Customize

Slide Distance: Edit translateY(20px) in .fade-in-section

Fade Speed: Change opacity 800ms transition duration

Slide Speed: Change transform 600ms transition duration

Threshold: Adjust threshold: 0.1 in JavaScript to change when elements start fading (0 = starts as soon as 1 pixel is visible, 1 = waits until fully visible)
